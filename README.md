
# RAID

RAID stands for Remittance Advice Interpreting Domain. The purpose of RAID is to interpret remittance advice in unstructured formats (e.g. PDF) and convert to clean, structured data (e.g. TXT).

### Goals:
- Minimal dependencies
- Speed

### Features:
- None

### Limitations:
- No multi-threading support yet - may add in the future
- Not compatible with Windows

### Dependencies:
- None

### Known Issues:
- Financial, non-claim-specific transactions are not interpreted or included in the output.
